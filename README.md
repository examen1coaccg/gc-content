# GC- Content

## Desripción
Este repositorio está enfocado al desarrollo de una herramienta que permita calcular y visualizar el contenido de Guanina-Citosina de secuencias genómicas.

## Motivación
En el ámbito de las ciencias genómicas conocer el grado de estabilidad de una secuencia genómica resulta de gran importancia. Esta es una de las actividades que se realiza día a día. Por ellos buscamos realizar un programa que permita llevar a cabo está tarea de manera más eficiente garantizando el análisis de secuencias de gran tamaño, asóí com opermitir la reproducibilidad de resultados.

## Requerimientos del sistema
- Python 3
- R
- Línea de comandos o algún IDE de desarrollo (Visual Studio Code)


## Ejecución
```python
python gc-content.py
```

## Estructura del repositorio

/docs: documentación del proyecto, como manuales de usuario
/lib: librerísa ocupadas para el desarrollo del software
/src: Scripts-código fuentes
/results: Ejemplo de resultados que validen el funcionamiento del software
/test: Casos de prueba del software
